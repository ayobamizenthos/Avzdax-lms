### Avzdax Academy LMS

Learning platform for Avzdax Academy students. Students take their course, watch lessons, do quizzes and assignments, join live classes and track their progress. Tutors run their own course. Admins handle users, courses and payments.

**Tech stack**

- Next.js (App Router, TypeScript)
- Tailwind CSS
- Supabase (Postgres, Auth, Storage, Realtime)
- Paystack
- Vercel

Live at [lms.avzdax.com](https://lms.avzdax.com)
